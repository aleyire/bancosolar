## Banco Solar
Sistema de transferencias realizado con un servidor con Node que utiliza PostgreSQL.
El sistema permite resgistrar nuevos usuarios con un balance inicial y basados en éstos, se puede realizar transferencias de saldos entre ellos.
Esta aplicación esta en desarrollo.

### Materia aplicada
- Conectar una base de datos PostgreSQL con Node.
- Realizar consultas DML con Node y el paquete pg.
- Realizar consultas TCL con Node y el paquete pg.
- Construir una API RESTful utilizando PostgreSQL para la persistencia de datos.
- Manejar errores.
- Manejar códigos de estado HTTP
